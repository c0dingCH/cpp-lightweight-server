### 主架构  
**Tcp** ： reactor 主从模式 ， 事件驱动模型  
**工具**：
  - 计时器 ：timer
  - 日志（前端同步，后端异步）： log
  - Cmake  
**简易http实现**： 静态文件的上传下载、get静态文件内容、简易账号密码的check
